# WhatsApp Setup

1. Install the optional group:
   ```bash
   pip install -r requirements-whatsapp.txt
   ```
2. Make sure Chrome or Chromium is installed.
3. Run the app and send a WhatsApp command with explicit confirmation.
4. On first use, scan the QR code in the browser window.

Contacts can be stored in an ignored `contacts.json` file at the project root:
```json
{
  "dad": "+15551234567"
}
```
