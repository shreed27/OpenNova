# Gemini Live Setup

1. Install the optional group:
   ```bash
   pip install -r requirements-gemini-live.txt
   ```
2. Add your API key:
   ```ini
   GEMINI_API_KEY=your_key_here
   ```
3. Grant camera and microphone permissions to your terminal app.
4. Run:
   ```bash
   python3 main.py
   ```
5. Ask JARVIS to start live vision.
